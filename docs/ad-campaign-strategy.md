# Threshold Google Ads Campaign Strategy

## Campaign Objective
Drive qualified demo bookings from campaign managers, political consultants, and progressive organizations looking for voter engagement tools.

---

## Budget & Realistic Goals

### Monthly Budget: $1,000 - $3,000

| Metric | Conservative ($1K/mo) | Mid ($2K/mo) | Aggressive ($3K/mo) |
|--------|----------------------|--------------|---------------------|
| Avg CPC (Search) | $4-6 | $4-6 | $4-6 |
| Monthly Clicks | 170-250 | 330-500 | 500-750 |
| Monthly Impressions | 5K-8K | 10K-16K | 16K-25K |
| CTR (Expected) | 3-5% | 3-5% | 3-5% |
| Landing Page Conv. Rate | 8-12% | 8-12% | 8-12% |
| **Demo Bookings/Month** | **14-30** | **26-60** | **40-90** |
| Demo-to-Customer Rate | 20-30% | 20-30% | 20-30% |
| **New Customers/Month** | **3-9** | **5-18** | **8-27** |

### Budget Allocation
- **70% Search Campaigns** ($700-$2,100): High-intent keyword targeting
- **20% Remarketing** ($200-$600): Retarget site visitors who didn't book
- **10% Brand** ($100-$300): Protect brand terms, capture direct searches

### Month-over-Month Goals

| Month | Budget | Target Demos | Focus |
|-------|--------|-------------|-------|
| 1 | $1,000 | 10-15 | Testing messaging, building data |
| 2 | $1,500 | 20-30 | Optimize winners, expand keywords |
| 3 | $2,000 | 30-45 | Scale proven ads, add remarketing |
| 4 | $2,500 | 40-55 | Full optimization, expand audiences |

## Landing Page Strategy

**Primary CTA:** Book a Demo (Cal.com integration already exists)

**Demo page should include:**
1. Clear headline: "See How AI Transforms Your Field Operation"
2. 60-second product video or GIF walkthrough
3. 3 key benefits with icons
4. Social proof (campaign testimonials, metrics)
5. Simple Cal.com embed for scheduling
6. "No credit card required. 15-minute demo."

## Key Messaging Themes

### Primary Value Prop
> "Built for the way campaigns actually run."

### Pain Points to Address
1. **Volunteer data entry is broken** — Volunteers hate forms, so data never gets entered
2. **Relational networks are untapped** — Volunteers know more voters than they realize
3. **Generic scripts don't work** — Voters respond to personalized, authentic conversations
4. **Campaign tools are overpriced** — Small campaigns can't afford enterprise software

### Messaging by Audience

| Audience | Pain Point | Message |
|----------|-----------|---------|
| Campaign Managers | Data quality & volunteer retention | "Volunteers talk, AI enters data. Clean exports every time." |
| Field Directors | Scaling personal outreach | "Map every volunteer's network. AI coaches every conversation." |
| Political Consultants | Client ROI on field | "10x more effective than mail. Real relational organizing at scale." |
| Progressive Orgs | Budget constraints | "50-75% cheaper than Mobilize. Free tier to start." |

## Tracking & KPIs

### Conversion Events (Track in Google Ads)
1. **Primary:** Demo booked (Cal.com completion)
2. **Secondary:** Contact form submitted
3. **Micro:** Demo page visited, pricing page visited, 60+ seconds on site

### Weekly Dashboard Metrics
- Cost per demo booking (target: $30-$75)
- Cost per click (target: $4-6)
- Click-through rate (target: 3-5%)
- Demo booking rate (target: 8-12%)
- Quality Score per keyword (target: 7+)

### Optimization Cadence
- **Daily:** Check spend pacing, pause underperformers
- **Weekly:** A/B test ad copy, adjust bids
- **Bi-weekly:** Review search terms report, add negatives
- **Monthly:** Full performance review, budget reallocation

## Setup Checklist

- [ ] Create Google Ads account at ads.google.com
- [ ] Set up billing with payment method
- [ ] Apply for Google Ads API developer token (takes 1-3 days)
- [ ] Create Google Cloud project and enable Google Ads API
- [ ] Set up OAuth2 credentials (Client ID + Secret)
- [ ] Configure conversion tracking (see `src/app/api/ads/conversion/route.ts`)
- [ ] Install Google Ads tag on thresholdvote.com
- [ ] Run setup script: `node scripts/google-ads/setup-campaigns.mjs`
- [ ] Verify conversion tracking is firing on demo bookings
- [ ] Launch campaigns (start paused, review, then enable)
